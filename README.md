# SOC Daily Workflow App

A browser-based alert triage tool built for daily CrowdStrike Falcon investigations.

🔗 **[Open the app](https://mperlingeiro.github.io/soc-workflow)**

---

## What it is

An interactive checklist-driven workflow I built for myself while completing the 
CyberXcel Applied Security Program. I use it every shift to stay structured when 
investigating real alerts generated by a live red team penetration tester.

This is my first project. More coming.

---

## What's inside

| Phase | Description |
|---|---|
| 1. Shift start | Orientation checklist before touching any alert |
| 2. Severity sort | How to prioritise the alert queue correctly |
| 3. Open an alert | What to read before taking any action |
| 4. Triage | Interactive true/false positive decision engine |
| 5. Respond | Step-by-step incident response for confirmed threats |
| 6. Document | Documentation checklist with UTC timestamp guidance |
| MITRE reference | Quick reference for the most common ATT&CK techniques |

---

## Tools and concepts covered

- CrowdStrike Falcon EDR
- MITRE ATT&CK Framework
- Incident triage and response
- Real-Time Response (RTR)
- Alert fatigue awareness
- SOC documentation standards
